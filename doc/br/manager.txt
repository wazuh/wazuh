Como usar o gerenciamento de clientes do servidor.

-A máquina cliente deve abrir a porta 1514 (por default) para enviará
 mensagens para o server. Somente o IP do servidor será liberado.

-A cada 10 minutos, o cliente envia uma notificação com o status para
 o servidor. Esta mensagem de status contem informações sobre o
 sistema do cliente e sobre os arquivos do diretório compartilhado.
 
-O servidor recebe a mensagem de status, atualiza o arquivo de status 
 do cliente e checa se há algum arquivo para ser enviado para o cliente.
 Se ouver, este conectará com o cliente e enviará o arquivo.

-Todas as mensagens são criptografadas.
