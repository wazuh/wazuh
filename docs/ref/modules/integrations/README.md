# Integrations

This section describes the Wazuh integrations with cloud services and third-party platforms for log collection, security monitoring, and threat detection.

## Cloud service integrations

- [GCP (Google Cloud Platform)](gcp.md) – Monitor Google Cloud audit logs via Pub/Sub and Cloud Storage.
- [AWS CloudTrail](aws-cloudtrail.md) – Collect and analyze AWS CloudTrail logs.
- [AWS CloudWatch Logs](aws-cloudwatch-logs.md) – Monitor AWS CloudWatch log groups.
- [AWS Security Hub](aws-security-hub.md) – Ingest findings from AWS Security Hub.
- [Amazon Security Lake](amazon-security-lake.md) – Retrieve security data from Amazon Security Lake.
- [Office 365](office365.md) – Collect audit logs from Microsoft Office 365.
- [GitHub](github.md) – Monitor GitHub organization audit logs.
- [Azure](azure.md) – Integrate with Azure Log Analytics, Storage, and Graph API.
- [Microsoft Graph Security API](microsoft-graph.md) – Retrieve security alerts and events from Microsoft Graph.
