
  - You have these installation options: manager or agent.

    - If you choose 'manager', you will be able to analyze all
      the logs, create e-mail notifications and responses,
      and also receive logs from systems running the 'agents'
      (from where traffic is sent encrypted to the manager).

    - If you choose 'agent'(client), you will be able to read
      local files (from syslog, snort, apache, etc) and forward
      them (encrypted) to the manager for analysis.

  - Choose 'manager' if you are setting up a log/analysis manager.

  - Choose 'agent' if you have another machine to run as a log
    manager and want to forward the logs to the manager for analysis.
    (ideal for webservers, database servers ,etc)

  - More information at:
    https://documentation.wazuh.com
